Accessible Palette Picker

This tool helps you choose an a11y-compliant color palette to use with Foundation by determining which colors meet WCAG 2.0 level AA contrast requirements.

Default Palette

The center column displays the default $foundation-palette colors. Each adjacent column is 3% lighter (to the left) or darker (to the right).
